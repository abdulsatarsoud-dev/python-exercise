# Python-Exercises

A collection of 4 small program I built while learning Python

### Projects Included

#### 1. **Calculator** 
 A simple calculator for +, -, *, / operations allow two input.
 
**Skills**: Function, conditionals and input handling

#### 2. **Grade System**
Converts a numeric score into a letter grade A-F based on standard range.

**Skills**: conditionals and input handling

#### 3. **Even or Odd Checker**
Check if a number entered by the user is even or odd.

**Skills**: Function, conditionals, basic logic and Modulo operator

#### 4. **Multiplication Table**
Generates a multiplication table for any number up to 10.

**Skills**: Formatted output, user input and  conditionals
